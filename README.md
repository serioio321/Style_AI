# StyleAI — Персональный стилист на базе ИИ

> Загрузи фото вещей из своего шкафа — получи готовые образы и рекомендации что докупить.

---

## Проблема

Большинство людей не знают, как сочетать вещи, которые уже есть в шкафу.

**В России:** по данным исследования Movinga (18 000 домохозяйств в 20 странах), россияне не надевали **53% вещей** из своего гардероба в течение года — и это один из лучших показателей в мире. ([FashionUnited, 2018](https://fashionunited.ru/novostee/moda/lyudi-ne-nosyat-polovinu-veschej-v-svoem-garderobe-issledovanie/2018082122914))

**В мире:** в Великобритании не используется **73% гардероба**, в Канаде и Швейцарии — **79%**, в Бельгии — **88%**. Средняя вещь надевается всего **7 раз** перед утилизацией. ([WRAP](https://www.wrap.ngo/media-centre/press-releases/nations-wardrobes-hold-16-billion-items-unworn-clothes-people-open-new), [FashionUnited UK](https://fashionunited.uk/news/fashion/people-do-not-wear-at-least-50-percent-of-their-wardrobes-according-to-study/2018081638356))

Проблема не в том, что вещей мало — а в том, что непонятно как их комбинировать. Существующие решения (Pinterest, IKEA AR) дают вдохновение, но не работают с твоим реальным гардеробом.

## Решение

StyleAI анализирует твои вещи с помощью компьютерного зрения и подбирает образы из того, что уже есть — плюс подсказывает, чего не хватает.

**Сценарий использования:**
1. Загружаешь фото вещей из шкафа
2. ИИ классифицирует каждую вещь: тип, цвет, стиль, сезон
3. Алгоритм подбирает образы с учётом совместимости цветов и стилей
4. Получаешь образы с фото вещей + описание от стилиста
5. Видишь чего не хватает в гардеробе

---

## Как это работает

```
Фото вещи
    ↓
LLaVA (vision модель) — классификация: тип / цвет / стиль / сезон
    ↓
SQLite — хранение гардероба
    ↓
Алгоритм совместимости — цветовые пары + стилевые группы → скоринг комбинаций
    ↓
LLaVA (текст) — описание образа как стилист
    ↓
Веб-интерфейс — карточки образов с фото вещей
```

---

## Стек

| Компонент | Технология |
|---|---|
| Backend | Python, FastAPI |
| ИИ (vision + текст) | Ollama + LLaVA 7B (локально) |
| База данных | SQLite + SQLAlchemy |
| Frontend | HTML/CSS/JS (без фреймворков) |
| Хранение фото | Локальная файловая система |

## Быстрый старт

### 1. Установи Ollama и скачай модель

```bash
# Скачай Ollama: https://ollama.com/download
ollama pull llava
```

### 2. Клонируй репозиторий и установи зависимости

```bash
git clone <repo>
cd PITCH_FASHION
pip3 install -r requirements.txt
```

### 3. Запусти

```bash
python3 -m uvicorn main:app --reload
```

Открой [http://localhost:8000](http://localhost:8000)

## Что можно добавить

### Парсинг офферов из интернета
Следующий шаг — к каждому образу автоматически подбирать похожие товары с маркетплейсов (Wildberries, Ozon, Lamoda) через парсинг или API. Пользователь видит не просто "тебе не хватает белой рубашки", а конкретные карточки товаров с ценами и ссылками — прямо внутри приложения.

### Персонализация под стиль
Анализ предпочтений пользователя по загруженным вещам → персональные рекомендации новых покупок с учётом уже сформированного стиля.

### Мобильное приложение
Снял шкаф на телефон → получил образы. Без компьютера.

## Монетизация

**Партнёрские размещения** — в блоке рекомендаций "что докупить" первыми показываются товары партнёров (маркетплейсы, бренды). Пользователь видит релевантный контент, партнёр платит за размещение или за переход.

**Сотрудничество с маркетплейсами** — интеграция напрямую с Wildberries / Ozon / Lamoda:
- Маркетплейс получает персонализированный трафик с высокой конверсией (пользователь уже знает что ищет)
- StyleAI получает комиссию с каждой покупки по реферальной ссылке

**White-label для брендов** — бренд встраивает StyleAI в своё приложение: покупатель загружает вещи бренда, получает образы, видит новые коллекции как рекомендации.
